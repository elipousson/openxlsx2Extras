# Common dxfs styles

`common_dxfs_styles` is a named list of dxfs styles created with
[`openxlsx2::create_dxfs_style()`](https://janmarvin.github.io/openxlsx2/reference/create_dxfs_style.html)
with bad, good, and neutral values.

## Usage

``` r
common_dxfs_styles
```

## Format

An object of class `list` of length 3.
